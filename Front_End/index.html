شبیه‌ساز NFA-λ و تبدیل به NFA

پروژه درس نظریه زبان‌ها و ماشین‌ها

تعریف ماشین

اسامی حالات را با فاصله یا ویرگول جدا کنید
حروف الفبا را وارد کنید (λ به صورت خودکار اضافه می‌شود)
برای انتخاب چندگانه کلیک کنید (تغییر رنگ نشان‌دهنده انتخاب است)

δ (تابع گذار)

از وضعیت نماد به وضعیت عملیات

📄 ورودی متنی

فرمت: Q = ... | Σ = ... | q0 = ... | F = ... | δ(...) = ...

❓ راهنمای فرمت ورودی

فرمت پیشنهادی:

Q = q0, q1, q2, q3
Σ = a, b
q0 = q0
F = q3
δ(q0, a) = q1
δ(q0, λ) = q2
δ(q1, b) = q3
δ(q2, a) = q3

نمادهای مجاز:

  • λ یا lambda برای انتقال پوچ
  • حروف انگلیسی و اعداد برای نام حالت‌ها
  • هر کاراکتر برای نمادهای الفبا (به جز λ)

نکات:

  • فقط یک حالت شروع مجاز است
  • چند حالت نهایی مجاز است
  • انتقال‌ها می‌توانند چندتایی باشند
